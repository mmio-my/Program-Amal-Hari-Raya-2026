# 🌙 Program Amal Hari Raya 2026 - Open Source Blueprint

Welcome to the official repository for the **Program Amal Hari Raya 2026**, organized by the [Malaysia Multimedia Influencers Organization (MMIO)](https://mmio.org.my/). 

As a digitally-native NGO, we believe in transparency and scalability. This repository serves as an open-source blueprint for executing a large-scale, high-impact Corporate Social Responsibility (CSR) event in Malaysia.

## 📌 Executive Summary
- **Event Name:** Program Amal Hari Raya 2026
- **Organizer:** Malaysia Multimedia Influencers Organization (MMIO) [PPM-033-10-10102024]
- **Target Beneficiaries:** 500 B40 families in Bandar Tun Razak, Kuala Lumpur
- **Date:** 18 April 2026
- **Official Portal:** [https://mmio.org.my/program_amal_2026/](https://mmio.org.my/program_amal_2026/)

## 💼 Sponsorship Framework (The Model)
To ensure sustainability, we structured our CSR funding into distinct tiers:
1. **Main Sponsor (RM 50,000):** Exclusive branding, VIP seating, and tax exemption pathways.
2. **Gold Sponsor (RM 30,000):** Prominent media coverage and on-site branding.
